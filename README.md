# Proyecto 3: Biblioteca Triple Peaks

La página web de la Biblioteca Triple Peaks es el tercer proyecto en el programa de Desarrollo web de TripleTen. Fue creado utilizando HTML y CSS, con base en un brief de diseño.

## Características del proyecto

- HTML5 semántico
- Flexbox
- Posicionamiento

## Funcionalidad

La página web de la Biblioteca Triple Peaks permite a los usuarios:

- Conocer la misión y propósito de la biblioteca.
- Consultar los libros recomendados por el staff.
- Ver los próximos eventos destacados.
- Acceder a un formulario de membresía con pasos claros para afiliarse.
- Navegar fácilmente entre secciones mediante enlaces de anclaje.
- Visualizar información de contacto, redes sociales y horarios en el pie de página.

Aunque el sitio es estático, está estructurado semánticamente para facilitar futuras integraciones dinámicas o interactivas.
